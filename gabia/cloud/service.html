g클라우드란?

g클라우드는 클라우드 기반으로 구현된 가상서버를 독립적으로 제공하는 서비스입니다.
CPU, RAM, HDD 모두 원하는 사양으로 설정할 수 있으며 높은 안정성을 제공합니다.
g클라우드는 Iaas형 클라우드로 자유롭고 탄력적인 서버 운용이 가능합니다.
프로비저닝 및 자동화 기술을 통해 인터넷이 연결된 곳이면 언제, 어디서나 바로 사용할 수 있으며
사용한 만큼 과금하는 방식으로 비용 절감 효과가 커 사용자에게 유리합니다.

클라우드에 관한 특징

경제성

초기 투자 비용 절감

  • 별도의 투자 없이 고성능 컴퓨팅 환경 사용
  • 확장성 보장을 위한 과잉 투자 불필요
  • 필요한 만큼만 사용, 사용한 만큼만 지불
  • VM별 트래픽 1TB 무료 지원
  • 공인 IP 무료 지원
  • 방화벽 무료 지원
  • 하드웨어 유지 및 교체 비용 불필요

편리성

운영 관리의 효율성 증대

  • 서버 현황 및 실시간 모니터링 툴 제공
  • 사용하기 쉬운 웹 관리 콘솔 제공
  • OS 및 플랫폼 등의 운영 환경 관리 부담 최소화
  • 고성은 서버 및 인프라를 제공하여 최적의 IT 환경 보장

신속성

IT 개발 환경 최적화

  • 다년의 호스팅 운영 노하우로 웹비즈니스에 최적화된 서비스 제공
  • 벤더 제품 사용이 아닌 클라우드 자체 기술력으로 제약 최소화
  • 자동차 기반의 동적 확장
  • 서버 이미지 기본 템플릿 제공
  • 신청 후 최소 1분 이내 인프라 사용 가능
  • 자체 기술력 확보로 확실한 기술

안정성

보안 및 안정성 극대화

  • 고가용성(HA)을 위한 물리적 인프라 구축
  • SLA로 클라우드 품질 보장
  • 보안접속 및 전송 암호화
  • 방화벽 기본 제공으로 외부 접속 보안 강화
  • 클라우드 인프라 전체 이중화 구성
  • 데이터 보호를 위해 OS 및 데이터 저장 장치 분리 구축
  • 보안 전문 인력 보유
  • 독립된 보안 관제 센터 운영
  • 365일 24시간 H/W 전문 관리자가 직접 관리

확장성

비즈니스 가치 상승

  • 탄력적인 인프라 운영 가능
  • 비즈니스 연속성 보장
  • 최대 10분 이내에 서버 생성 및 사용 가능
  • 최적화된 가용성 및 확장성 보장
  • 비즈니스에 필요한 서버 사양 직접 설정
  • 부하 분산을 위한 로드밸런싱 기능 제공
  • 철저한 장애 대비 시스템 구축
  • 스냅샷 및 이미지 생성 기능 제공

인프라를 운영하세요

인증 받은 클라우드 품질

국내 중소기업 최초로 ‘클라우드 서비스 보안 인증’과
‘클라우드 서비스 품질 인증(5Level)’을 획득하였으며,
‘클라우드 서비스 확인제’ 검증을 통과했습니다.

SLA(품질 보장제)

높은 수준의 월 가용률 보장으로
안정성을 극대화합니다.

빠른 서버 설치 및 배포

원하는 OS 이미지와 사양만 선택하면
최대 10분 이내에
서버 생성 및 이용이 가능합니다.

고성능 인프라 임대

서버, 네트워크, 스토리지의
개별 구성으로 안정적인
인프라 환경을 구축했습니다.

탄력적 서버 운영

비즈니스 상황에 맞춰
서버 자원을 배분하여 운영하며,
사용한 만큼 지불하는
후불제 서비스를 제공합니다.

클라우드 전문 기술지원

클라우드 전문 영업자의 1:1 컨설팅과
전문 SE 및 개발 그룹의
365X24 기술 지원을 제공합니다.

클라우드 운영에 필요한 모든 기능을 보장합니다

트래픽

서버(VM)마다 월 1TB의
트래픽을 제공합니다.

사설 IP와 공인 IP

사설단 구성을 위한 서버별 사설 IP와
웹 서비스 제공을 위한 공인 IP를
ID당 1개, 무료 제공합니다.

네임서버

쉬운 DNS 관리 기능으로
호스트를 최대 100개까지
등록할 수 있습니다

방화벽

보안 사고 예방 및 빠른 대응을 위한
실시간 정책 설정 기능을 제공합니다.

모니터링

시스템 성능, 네트워크 사용 현황 등을
한 눈에 실시간으로 볼 수 있는
GUI를 제공합니다.

모니터링 알람

CPU, Memory, 네트워크 등
특정 요소의 사용량 초과 시
알람을 발송하여 예기치 못한
문제 발생을 사전에 차단합니다.

서버 시작, 중지, 재시작, 삭제, 불륨 장착 등
언제든지 클라우드 운영에 필요한 기능을
콘트롤 할 수 있습니다.
